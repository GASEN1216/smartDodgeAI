Smart Dodge AI v0.2.1.0

[h2]新增功能 (New Features)[/h2]
*   新增墨水防护饰品系统：包含5个基础饰品和1个高级饰品，可以有效减少敌人墨水喷射次数。
*   新增愚人节彩蛋功能：骷髅专用模式，仅在4月1日可用，包含Megalovania音乐效果。
*   新增墨水喷射闪避机制：敌人可以通过喷射墨水弹幕来闪避攻击。

[h2]New Features[/h2]
*   Added Ink Protection Accessory System: Includes 5 basic accessories and 1 advanced accessory that effectively reduce enemy ink splash count.
*   Added April Fools Easter Egg: Skeleton-only mode available only on April 1st, includes Megalovania music effects.
*   Added Ink Splash Dodge Mechanism: Enemies can dodge attacks by spraying ink projectiles.

[h2]改进 (Improvements)[/h2]
*   更新了README文档中的饰品描述，使其更加清晰易懂。
*   优化了墨水喷射效果的视觉效果和音效。

[h2]Improvements[/h2]
*   Updated accessory descriptions in README documentation to be more clear and understandable.
*   Optimized visual effects and sound effects for ink splash mechanics.

Smart Dodge AI v0.2.0.2

[h2]新增功能 (New Features)[/h2]
*   新增瞬移视觉预警：敌怪瞬移前，目标位置会提前0.5秒出现残影，提供躲避时机。
*   新增“瞄准芯片”饰品，可以降低敌人的闪避几率，由任意职业徽章合成。

[h2]New Features[/h2]
*   Added Teleport Visual Cue: Before an enemy teleports, an afterimage appears at the target location 0.5 seconds in advance, providing a window to dodge.
*   Added "Targeting Chip" accessory, which reduces enemy dodge chance and is crafted from any class emblem.

Smart Dodge AI v0.2.0.1

[h2]新增功能 (New Features)[/h2]
*   新增战术性瞬移功能：敌怪在闪避后，若正在冲向玩家，则会瞬移到玩家身后并发起追击，整个过程可通过配置菜单开关和调整。

[h2]New Features[/h2]
*   Added tactical teleportation: After dodging, if an enemy is charging, it will teleport behind the player to continue the chase. This feature can be toggled and adjusted in the config menu.

[h2]改进 (Improvements)[/h2]
*   重构了配置菜单，为不同类型的设置添加了标题进行分类。
*   为所有配置选项添加了更详细的中文和英文说明，以提高易用性。 

[h2]Bug Fixes[/h2]
* Fixed an issue where NPCs would cause summons to disappear when dodging their attacks.

[h2]Improvements[/h2]
* Restructured the configuration menu and added titles to categorize different types of settings.
* Added more detailed Chinese and English descriptions for all configuration options to improve usability.

Smart Dodge AI v0.2.0

[h2]新增功能 (New Features)[/h2]
*   现在可以按伤害类型（远程、魔法、召唤、近战）设置独立的闪避率。
*   现在可以为特定的弹幕类型设置独立的、更高优先级的闪避率。

[h2]Bug 修复 (Bug Fixes)[/h2]
*   修复了 NPC 闪避召唤物攻击时，会导致召唤物自身消失的问题。

[h2]改进 (Improvements)[/h2]
*   重构了配置菜单，为不同类型的设置添加了标题进行分类。
*   为所有配置选项添加了更详细的中文和英文说明，以提高易用性。 

[h2]New Features[/h2]
* You can now set independent dodge rates by damage type (ranged, magic, summon, melee).
* You can now set independent, higher priority dodge rates for specific barrage types.

[h2]Bug Fixes[/h2]
* Fixed an issue where NPCs would cause summons to disappear when dodging their attacks.

[h2]Improvements[/h2]
* Restructured the configuration menu and added titles to categorize different types of settings.
* Added more detailed Chinese and English descriptions for all configuration options to improve usability.
